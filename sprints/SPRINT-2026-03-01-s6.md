# Sprint 6 — Forge: Chord Suggestions + Text Export
*2026-03-01*

## Goal
Close the guidance gap in the Forge: given a chord, suggest the other diatonic chords in a selected key. Add text file download as an alternative to clipboard-only export.

## Decisions
- Key selector sits in `.forge-controls` alongside tuning — `forgeKeyId` state mirrors `forgeTuningId` pattern.
- Suggestions use root-note matching only (`dc.root === chordRoot`) so `Am7` still highlights the `Am` diatonic slot correctly.
- `buildForgeTabText()` extracted from `exportForgeTab()` — both clipboard and download call it.
- `.forge-chord-panel` changed to `flex-direction: column` to accommodate suggestions below chord identification.
